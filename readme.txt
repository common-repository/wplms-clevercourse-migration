WPLMS Clevercourse Migrate helps users to migrate all Clevercourse courses to WPLMS courses

== Description ==
WPLMS Clevercourse Migrate plugin is a free plugin developed to help users to migrate all of their Clevercourse courses to WPLMS courses. The users do not have to worry a lot and perform various steps to migrate their courses. All this is done by a simple click. With the courses all of the quizzes and the units will be migrated successfully with just one click.

Tutorial On how to setup and get started : <a href="https://vibethemes.com/documentation/wplms/knowledge-base/clevercourse-good-lms-migration-to-wplms/">link</a>

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'WPLMS Clevercourse Migrate'
3. Activate WPLMS Clevercourse Migrate from your Plugins page. 

= From WordPress.org =

1. Download WPLMS Clevercourse Migrate.
2. Upload the 'wplms_clevercourse_migrate' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate WPLMS Clevercourse Migrate from your Plugins page.


== Frequently Asked Questions ==

= Does it require Clevercourse theme? =

Yes, The plugin needs the Clevercourse theme to be activated to work.

= Do I need the WPLMS theme to begin? =

No, You need the clevercourse theme to begin but after migrating you need the WPLMS theme to see all the changes. 

= Can I revert changes if I don't like it after migrating? =

Yes, after migration you can check all the courses and if you feel its not correct and again wants to move to Clevercourse theme, then you can simply deactivate the WPLMS theme and again activate the Clevercourse theme to get all the courses in the Clevercourse theme LMS.
